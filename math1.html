Exercises

my-inc

Clojure has function inc, which accepts one number and returns that number increased by one. Write function my-inc, which does the same. Don't use built-in inc.

(defn my-inc [x]
  (+ x 1))
New function unclocked! inc

my-dec

Clojure has function dec, which accepts one number and returns that number decreased by one. Write function my-dec, which does the same. Don't use built-in dec.

(defn my-dec [x]
  (- x 1))
New function unclocked! dec
