my-abs

Clojure has function abs, which accepts one number and returns its absolute value. Write function my-abs, which does the same. Don't use built-in abs.

Tests with the expected results: 

(my-abs 2)
=> 2

(my-abs 0)
=> 0

(my-abs -2)
=> 2

(defn my-abs [x]
  (if (neg? x)
    (- x)
    x))

sgn

Write function sgn, which returns signum of given number:

Tests with the expected results: 

(sgn 5)
=> 1

(sgn 0)
=> 0

(sgn -5)
=> -1


(defn sgn [x]
  (cond (pos? x) 1
        (zero? x) 0
        :else -1))

Logical conjunction

Write a function lc with two arguments. Each argument will be number with value 0 or 1 (this represents false or true). This function returns true or false, based on truth table of logical conjunction.

Tests with the expected results: 

(lc 0 0)
=> false

(lc 0 1)
=> false

(lc 1 0)
=> false

(lc 1 1)
=> true

(defn lc [a b]
  (and (not (zero? a))
       (not (zero? b))))
