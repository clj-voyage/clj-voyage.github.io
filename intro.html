Intro

Exercises

In each chapter, you'll be introduced to some new functions for solving exercises. When you encounter function unknown to you:

  • Read about it in your favourite Clojure book
  • Call (doc ...) and read its documentation
  • Read about it in Clojure Docs and study some examples here
  • Run some experiments in REPL to see, if you can predict behaviour of this function. What kind of arguments this function accepts, what is the type of result?
For every exercise, you need only functions introduced to you in current or one of the previous chapters. Don't use any additional functions, it can lead you on the wrong track.

Editor

Don't use some online Clojure editor without REPL, it will spoil your experience- pick one of these and get familiar with it:

  • Emacs + CIDER
  • IntelliJ IDEA + Cursive
  • Visual Studio Code + Calva
For more info, see Clojure Guides: Editors.
