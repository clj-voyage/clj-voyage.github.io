Exercises

For these exercises, you have to require Clojure math library. Change your namespace definition and evaluate it again: 
(ns my-project-name.core
  (:require
    [clojure.math :as math])
  (:gen-class))
What is the value of these expressions? Evaluate in REPL:
  • (apropos "pow")
  • (apropos "sinus")
  • (apropos "sin")
  • math/PI
  • (math/sin (math/to-radians 90))

circle area, circle circumference

Write a function circle-area, which accepts one number and returns area of circle with that radius.

Write function circle-cf, which accepts one number and returns circumference of circle with that radius.

You can use functions from Math library, that you found in the documentation or with apropos.

Tests with the expected results: 

(circle-area 10)
=> 314.1592653589793

(circle-cf 10)
=> 62.83185307179586

(defn circle-area [r]
  (* math/PI r r))

(defn circle-cf [r]
  (* 2 math/PI r))

hypotenuse

Write a function hypotenuse, which accepts two numbers (lengths of two sides in a right-angled triangle) and returns the length of the hypotenuse, using a Pythagorean theorem.

You can use functions from Math library, that you found in the documentation or with apropos.

Tests with the expected results: 

(hypotenuse 3 4)
=> 5.0

(defn hypotenuse [a b]
  (math/sqrt (+ (* a a)
                (* b b))))
